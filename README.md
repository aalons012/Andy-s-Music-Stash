# Andy's Music Stash

Andy's Music Stash is a responsive music library website built with HTML, CSS, and JavaScript. The site serves as a curated collection of favorite songs, albums, artists, and playlists across multiple genres and languages. Users can explore different music categories, discover random songs, and browse album artwork through an interactive slideshow.

## Features
🎵 Random song generator

🎸 Album, artist, and playlist pages

🎼 Genre-specific collections (Rock, Metal, Rap, Indie, Corridos, etc.)

🖼️ Album artwork slideshow

🌎 Diverse music selection spanning multiple languages and styles

📱 Simple and responsive web interface

## Technologies Used
* HTML5
* CSS3
* JavaScript
* Media assets (MP3 and image files)
